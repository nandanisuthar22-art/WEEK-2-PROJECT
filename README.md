🌧️ Rainfall Pattern Analysis

A comprehensive data analysis project that simulates, cleans, analyzes, and visualizes rainfall data across major Indian cities using Python. This project demonstrates real-world data handling, exploratory data analysis (EDA), and interactive visualizations.

📌 Project Overview

This project focuses on analyzing rainfall patterns using a synthetically generated messy dataset (2500 rows). It walks through the full data pipeline:

Data Generation (with inconsistencies)
Data Cleaning & Preprocessing
Exploratory Data Analysis (EDA)
Interactive Visualizations
🚀 Features
✅ Realistic messy dataset simulation
✅ Robust data cleaning techniques
✅ Statistical analysis of rainfall, temperature, and humidity
✅ Seasonal & monthly trend insights
✅ Interactive visualizations using Plotly
✅ Correlation analysis between environmental factors
🛠️ Tech Stack
Python
Pandas – Data manipulation
NumPy – Numerical operations
Matplotlib / Seaborn – Static visualizations
Plotly – Interactive charts

⚙️ Workflow
1️⃣ Data Generation
Created 2500 rows of synthetic rainfall data
Includes:
Multiple cities (Mumbai, Delhi, Bangalore, Chennai, Pune)
Seasons & rainfall types
Intentional inconsistencies (case issues, formats, missing values)

2️⃣ Data Cleaning
Standardized date formats
Cleaned inconsistent categorical values
Handled missing/null values
Normalized rainfall categories

3️⃣ Data Analysis
Descriptive statistics
Monthly rainfall trends
Seasonal comparisons
City-wise rainfall distribution
4️⃣ Visualizations
📈 Monthly Rainfall Trend (Interactive)
📊 City-wise Rainfall Distribution (Box Plot)
🌦️ Seasonal Rainfall Comparison
🗺️ Rainfall Heatmap
🔗 Correlation Matrix

📊 Key Insights
Monsoon season shows significantly higher rainfall
Coastal cities (e.g., Mumbai, Chennai) receive more rainfall
Strong correlation between humidity and rainfall
Seasonal variations heavily impact rainfall distribution
